# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-04-28

### Changed

- Bumped OpenOrchestrator to 3.*
- Switched main.py bootstrap to uv

## [1.1.2] - 2025-09-08

### Fixed

- Missing version number on Selenium package.

### Added

- Robot is now using queue elements.
- Added event logging.

## [1.1.1] - 2025-04-25

### Fixed

- Fixed crash when no contact information is available.

## [1.1.0] - 2024-10-07

### Added

- Queue elements are now created for every phone number checked, for statistical purposes.

## [1.0.3] - 2024-09-26

### Fixed

- Added files for github flow checks

## [1.0.2] - 2024-09-25

### Fixed

- Robot now uses it's own set of credentials for logging into eFlyt

## [1.0.1] - 2024-09-24

### Fixed

- Updated README.md

## [1.0.0] - 2024-09-04

- Initial release
